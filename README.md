# financial-core-microservices

> Learning journey through microservices architecture by building a short banking system with Accounts, Loans, and Cards services.

## 🎯 What I'm learning
- Java + Spring Boot microservices
- Spring Cloud (Config, Discovery, Gateway)
- Docker containerization
- Kubernetes orchestration
- Helm packaging
- Microservices Security

## 🏦 Domain Services
| Service | Responsibility |
|---------|---------------|
| Accounts | Customer account management |
| Loans | Loan products and processing |
| Cards | Credit/debit card management |
